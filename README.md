## The ACI badge

Every ACI here carries the full complement, in the format the standard specifies:

- **`.carbon` → TIFF** — the warm sigil, archival and lossless (`II*\0`)
- **`.silicon` → PNG** — the cool sigil, for the web
- plus `.agent` (the persona) · `.spun` (the five W's) · `.moniker` (the tokenized mythos) · `.1099` (the credit ledger) · `manifest.dlw.json`

The main **URSULA** badge — `⟦URSULA:U1:60c7c7⟧` — lives in [`ursula.dlw/`](ursula.dlw/).

## On scope

Earthsea is fantasy; the Hainish Cycle is science fiction; Le Guin spent a life refusing the wall between them — so this catalogues **both**, plus the essays. The works and characters are © the Ursula K. Le Guin estate; the personas are catalogued personifications under the DLW standard — bibliographic commentary, not original creations. Credit for the catalogue returns to the human.
